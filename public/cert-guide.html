🔒 Небезопасное соединение

Не удаётся открыть страницу по защищённому соединению

Похоже, на устройстве отсутствует или отозван корневой сертификат Минцифры России. Без него браузер не может проверить сертификат сайта. Установите сертификат по инструкции ниже — после этого приложение снова откроется.

Скачать сертификаты на Госуслугах

Windows

  1. Скачайте оба сертификата на gosuslugi.ru/crt: «Russian Trusted Root CA» и «Russian Trusted Sub CA».
  2. Дважды кликните по файлу сертификата → Установить сертификат.
  3. Выберите расположение «Локальный компьютер» и нажмите «Далее».
  4. Отметьте «Поместить все сертификаты в следующее хранилище» → «Доверенные корневые центры сертификации».
  5. Завершите установку и повторите для второго сертификата.
  6. Полностью закройте и снова откройте браузер.

macOS

  1. Скачайте оба сертификата с gosuslugi.ru/crt.
  2. Откройте приложение «Связка ключей» (Keychain Access), выберите связку «Система».
  3. Перетащите файлы сертификатов в окно связки ключей.
  4. Дважды кликните по каждому сертификату → раздел «Доверие» → «Всегда доверять».
  5. Перезапустите браузер.

iOS / iPadOS

  1. Откройте gosuslugi.ru/crt в Safari и скачайте профиль сертификата.
  2. Перейдите в Настройки → Основные → VPN и управление устройством и установите загруженный профиль.
  3. Затем откройте Настройки → Основные → Об этом устройстве → Доверие сертификатов.
  4. Включите полное доверие для сертификатов Russian Trusted CA.
  5. Перезапустите Safari.

Android

  1. Скачайте сертификаты с gosuslugi.ru/crt.
  2. Откройте Настройки → Безопасность → Шифрование и учётные данные → Установка сертификатов → Сертификат ЦС.
  3. Выберите скачанный файл и подтвердите установку.
  4. Рекомендуется использовать Яндекс Браузер — в нём сертификаты Минцифры уже встроены.

Официальный источник сертификатов — портал Госуслуг. Устанавливайте сертификаты только с него.
